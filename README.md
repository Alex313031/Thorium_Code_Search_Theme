codesearch-theme

Ideal for Chromium devs!

Simple extension to help you rest your eyes while navigating the source tree.

Supporting themes:
- Tomorrow themes

Shortcuts:
- Next theme: CTRL + SHIFT + L.
- Previous theme: CTRL + SHIFT + K.
